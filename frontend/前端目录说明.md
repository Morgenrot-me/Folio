# 前端架构说明

## 1. 目录作用
存放智能图片分类的全部前端代码，包含基于 Flutter 的跨平台界面和 Dart 业务逻辑代码。负责 UI 展示、用户交互及底层（本地/后端）调用。

## 2. 文件列表
本目录由 Flutter 标准结构生成，包含：
- `lib/main.dart`: 应用程序入口
- `pubspec.yaml`: 依赖配置文件

## 3. 技术说明
本项目采用 Flutter 和 Material Design 3 进行构建，通过 tflite_flutter 执行本地模型推理。

## 4. 使用示例
进入该目录下，通过以下命令启动调试：
```bash
cd frontend
flutter run
```

## 5. 扩展方式
- 新页面/功能模块应组织在 `lib/features/` 目录下。
- 新增依赖需要在 `pubspec.yaml` 中声明。

## 6. 注意事项
- 每次修改代码后务必提交 Git。
- 前后端或数据库字段有变更时请保持多处同步处理。
- 请保持这里的文档内容与实际文件结构时刻同步更新。
